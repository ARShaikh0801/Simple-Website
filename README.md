## 🚀 About The Project

**Simple Website** is a lightweight, high-performance landing page project demonstrating modern web design techniques without relying on bulky frameworks. It features a stunning glassmorphic UI, smooth CSS animations, and a fully responsive layout.

### ✨ Key Features

- **Pristine Design:** A clean, contemporary interface featuring beautiful gradients and glassmorphism.
- **Fully Responsive:** Flawless rendering on desktops, tablets, and mobile devices, complete with a custom CSS hamburger menu.
- **Modern Footer:** A comprehensive footer containing quick links, brand information, and social media connectivity.
- **Fast Performance:** Pure HTML, CSS, and minimal vanilla JavaScript ensure instant load times.
- **No Dependencies:** Built entirely with core web technologies.

## 🛠️ Built With

* [HTML5](https://developer.mozilla.org/en-US/docs/Web/HTML)
* [CSS3](https://developer.mozilla.org/en-US/docs/Web/CSS) (Flexbox, Grid, Custom Variables, Animations)
* Vanilla [JavaScript](https://developer.mozilla.org/en-US/docs/Web/JavaScript) 

## 💻 Getting Started

This project is completely static and requires no build steps or package installations. 

### Prerequisites

All you need is a modern web browser (Chrome, Firefox, Safari, Edge).

### Installation & Usage

1. Clone the repo
   ```sh
   git clone https://github.com/your_username/simple-website.git
   ```
2. Navigate into the project directory
   ```sh
   cd simple-website
   ```
3. Open `index.html` in your favorite web browser!
   - *Pro-tip:* You can also use [Live Server](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer) in VS Code to run the site with hot-reloading.

## 📱 Responsive Previews

The website fluidly adapts to different screen sizes. Try resizing your browser window or viewing it on your phone to see the mobile navigation and stacked grid layouts in action!
